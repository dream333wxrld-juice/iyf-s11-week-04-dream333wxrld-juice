## Project Description
Built interactive JavaScript applications covering core concepts including variables, data types, functions, control flow, arrays, and objects. Created a functional calculator and a student grade management system demonstrating data manipulation and object-oriented programming.

## Technologies Used
- HTML5
- JavaScript

## Features
- Interactive calculator with addition, subtraction, multiplication, division, modulus, and power operations
- Student grade tracker system with student management
- Array methods: map, filter, reduce, find, some, every
- Object manipulation and iteration
- Control flow with loops and conditionals
- String reversal and palindrome checking

## How to Run
1. Clone this repository
2. Open `index.html` in your browser
3. Open the browser console to see output from main.js

## Lessons Learned
Learned how to use array methods effectively for data transformation and filtering. Understood how to work with objects, access nested properties, and create methods within objects. Gained proficiency with arrow functions and callbacks.

## Challenges Faced
Initially struggled with understanding the difference between array methods like map vs filter. Also took time to fully grasp how reduce works with accumulators. Solved these by practicing with multiple examples and building the grade tracker project.
