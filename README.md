## Project Description
Built interactive JavaScript applications covering core concepts including variables, data types, functions, control flow, arrays, and objects. Main projects include an interactive calculator and a student grade tracker.

## Technologies Used
- HTML5
- JavaScript

## Features
- Interactive calculator with basic math operations
- Student grade tracker with data manipulation
- FizzBuzz program
- String reversal function
- Largest number finder
- Duplicate remover
- Palindrome checker

## How to Run
1. Clone this repository
2. Open `index.html` in your browser
3. Open the browser console to see output

## Lessons Learned
Learned how JavaScript handles variables, functions, arrays, and objects. Understood the difference between let and const, and how to use array methods like map, filter, and reduce.

## Challenges Faced
Understanding the difference between loose and strict equality operators. Also getting comfortable with arrow functions and when to use them.
